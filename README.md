# SimpText
SimpText 是一款簡單但功能強大的文本編輯器，專為輕量級編輯需求而設計，類似於 Notepad++，適用於 Linux,Windows 平台。

## 主要功能
- 輕量快捷：啟動迅速，佔用資源少。
- 語法高亮：支援多種程式語言，提高編輯效率。
- 簡潔界面：專注內容，避免視覺干擾。
- 多標籤頁支持：方便管理多個文件。（還正在實現中）
- Linux 原生支援：可直接打開，易於使用。
